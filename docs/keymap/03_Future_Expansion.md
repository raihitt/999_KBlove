# 将来的な拡張性・コンボ案メモ

今後の keymap 拡張に向けたアイデア集です。

## 1. 編集キー系コンボ（定番）

ホームポジションを崩さずに一般的な編集操作を行うための設定案。

| コンボ | 出力 | 理由・使いどころ |
| :--- | :--- | :--- |
| `Q + W` / `J + F` | **Escape** | 左手ホームポジションで Esc（Vim等に最適） |
| `A + S` / `F + D` | **Backspace** | 最も頻度の高い削除操作の効率化 |
| `S + D` | **Delete** | 右側削除 |
| `X + C` | **Ctrl + C** (Copy) | ホームポジション近くでコピー |
| `C + V` | **Ctrl + V** (Paste) | ホームポジション近くでペースト |
| `Z + X` | **Ctrl + Z** (Undo) | 直感的な取り消し操作 |
| `X + Z` | **Ctrl + Y** (Redo) | やり直し |

## 2. 記号・括弧系コンボ

小指の負担を減らし、ホームポジションで完結させる案。

| コンボ | 出力 | 備考 |
| :--- | :--- | :--- |
| `, + .` | **( )** | 括弧の対入力 |
| `M + ,` | **{** | 中括弧（開始） |
| `. + /` | **}** | 中括弧（終了） |
| `J + K` | **[** | 角括弧（人差し指＋中指） |
| `K + L` | **]** | 角括弧（中指＋薬指） |
| `U + I` | **<** | タグ入力等 |
| `I + O` | **>** | タグ入力等 |
| `H + J` | **-** | ダッシュ（Markdown等） |
| `J + K` | **_** | アンダースコア |

## 3. レイヤー切り替え・モディファイア系

ホームローモディファイアの補完または代替案。

| コンボ | 出力 | 備考 |
| :--- | :--- | :--- |
| `Space + F` / `J` | **Ctrl** | 親指＋人差し指で Ctrl |
| `Space + D` / `K` | **Shift** | |
| `Space + S` / `L` | **Alt** | |
| `Space + A` / `;` | **GUI (Cmd/Win)** | |
| `T + G` | **NAV レイヤー** | 矢印・Home/End 等 |
| `B + N` | **SYM レイヤー** | 記号集中入力 |

## 4. 生産性向上マクロ

よく使う定型文や操作の一発呼び出し案。

- **人差し指ホーム + 中指ホーム**: `https://`
- **`Q + A`**: `sudo `
- **`Z + C`**: `console.log(` （プログラミング向け）
- **`, + M`**: `→` （Markdown 引用・矢印）
- **`. + ,`**: `git commit -m "`

## 5. 押しやすさランキング（運指設計の指針）

1. **人差し指 + 中指**: 一番押しやすい（最優先）
2. **中指 + 薬指**: 良好
3. **人差し指 + 薬指**: 少し難しい
4. **親指 + 人差し指**: 特殊操作に最適
5. **薬指 + 小指**: 極力避ける
