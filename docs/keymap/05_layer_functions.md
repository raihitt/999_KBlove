# レイヤー機能概要

このドキュメントでは、各レイヤーの役割と呼び出し方法についてまとめています。

## レイヤー一覧

| No | 名称 | 主な機能 | 呼び出し方法 |
|:---:|:---|:---|:---|
| 0 | DEFAULT | 通常タイピング (QWERTY) | 基本レイヤー |
| 1 | MOUSE | マウスボタン操作 (K=左, L=右) | トラックボール操作 (AML) |
| 2 | SCROLL | 縦横自在スクロール | コンボ `K + L` または `lt 2 /` |
| 3 | ARROW | カーソル移動、ウィンドウ操作 | `lt 3 BSPC` または マウス内 `mo 3` |
| 4 | NUM | 数字入力、テンキーレイアウト | `lt 4 SPACE` |
| 5 | FUNCTION| F1 〜 F12 キー | `lt 5 ESC` |
| 6 | SYSTEM | BT切替、システム操作 | `mo 6` |
| 7 | VSCROLL | 縦専用スクロール (X軸抑制) | コンボ `I + O` |

---

## 各レイヤーの詳細

### 0: DEFAULT
- 標準の QWERTY 配列です。
- 親指キーにレイアウト変更（`lt`）が集中しています。

### 1: MOUSE
- **AML (Auto Mouse Layer)** により、ボールを動かすと自動で遷移します。
- **K**: 左クリック (MB1)
- **L**: 右クリック (MB2)
- **J + K (Combo)**: ミドルクリック (MB3)
- **親指 (BSPC位置)**: ARROWレイヤーへの `mo` 遷移（マウス移動しながらの方向キー操作用）

### 2: SCROLL
- トラックボールがスクロールホイールとして動作します。
- `K + L` 同時押し、または右下 `/` キーの長押しで利用可能です。

### 3: ARROW
- 左手側に Tab 切替や Cmd 有りの矢印、右手側に純粋な矢印キーを配置。
- `BSPC` キーの長押しで呼び出します。

### 4: NUM
- 右手側にテンキーを模した配置。
- `SPACE` キーの長押しで呼び出します。

### 5: FUNCTION
- `1` 〜 `0` の行が F1 〜 F10 に対応。 F11, F12 は別途配置。

### 6: SYSTEM
- Bluetooth のペアリング（BT_SEL）やプロファイル切替、基本的なシステムショートカットを配置。

### 7: VSCROLL (縦スクロール専用)
- **I + O** の同時押しで起動します。
- X軸の入力を1/16に減衰させているため、意図しない横揺れを抑えて縦スクロールに集中できます。

## 実装済みコンボ一覧

| コンボ | キー位置 | 動作 | 備考 |
| :--- | :---: | :--- | :--- |
| `J + K` | Pos 18, 19 | **ミドルクリック** (MB3) | 人差し指＋中指 |
| `K + L` | Pos 19, 20 | **スクロールレイヤー** (Layer 2) | 中指＋薬指 |
| `I + O` | Pos 7, 8 | **縦スクロール専用** (Layer 7) | 右手最上段（定番） |
