OpenSpec-driven Python project delivering an IRBANK financial metrics scraper. Specs and change logs live in `openspec/`; active changes are staged under `openspec/changes/` while canonical specs reside in `openspec/specs/`. Implementation code (when present) belongs under `irbank-scraper/src/` with tests in `irbank-scraper/tests/`. The scraper fetches IRBANK HTML, normalizes currencies/percentages, computes derived metrics (e.g., cash conversion cycle), and exports 10 fiscal years of confirmed data to CSV.
